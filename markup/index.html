Селектори

Вони бувають:

  • Універсальний, який відповідає будь-якому елементу веб-сторінки, що встановлює один стиль для всіх цих елементів, задати шрифт або накреслення тексту. Синтаксис: * {...}
  • Теговий задає стилі для усіх елементів даного тега. Синтаксис: a {...}
  • Класовий задає стилі для одного або більше елементів, але з однаковим класом, які знаходяться в разних місцях сторінки. Синтаксис: .a {...}
  • Селектор індефікатора який задає стилі для певного елементу, бо знаємо, що id є унікальним. Синтаксис: #a {...}
  • Селектор нащадка який дозволяє задати стилі до елементів, які розташовані всередині елемента-контейнера. Синтаксис: a b { ... }
  • Дочірний селектор задає стилі до елементів, які є прямими нащадками. Синтаксис: a > b { … }
  • Сусідній селектор обирає елемент, який знаходиться безпосередньо після вказаного елементу. Синтаксис: a + b { … }
  • Родинний селектор обирає усі елементи, які знаходяться після вказаного елементу. Синтаксис: a ~ b { … }
  • Селектор атрибута задає стилі до елементів на основі ім'я атрибута або значення атрибта. Синтаксис: a[attr=”value”] { … }

Що таке CSS? формальна мова опису зовнішнього вигляду документа, написаного за допомогою мови розмітки.

CSS привносить життя HTML-документ, вибираючи шрифти, застосовуючи кольори, визначаючи відступи, позиціонуючи елементи, анімуючи взаємодії та багато іншого.

Основні види селекторів

  • * – різні елементи.
  • div – елементи з таким тегом.
  • #id – елемент з даним id.
  • .class – елементи з таким класом.
  • [name="value"] – селектори на атрибут.
  • «псевдокласи», інші різні умови на елемент.

Каскадність CSS – це механізм, завдяки якому елемент HTML-документа може застосовуватися більш ніж одне правило CSS. Правила можуть виходити з різних джерел: із зовнішньої та внутрішньої таблиці стилів, від механізму успадкування, від батьківських елементів, від класів та ID, від селектора тега, від атрибуту style тощо. система пріоритетів: зрештою застосовується той стиль, який походить від джерела з вищим пріоритетом.1) специфічність ("вага" селектора):_

Коли селектор складається з кількох інших селекторів, необхідно порахувати їхню загальну вагу. Ось як обчислюється пріоритет: за кожен селектор додається 1 у відповідну комірку. В інших осередках стоять нулі. Щоб отримати загальну вагу, необхідно «склеїти» всі числа в осередках.

Комбінування та угруповання селекторів

Для більш точного відбору елементів форматування можна використовувати комбінації селекторів:

a[href][title] { color: red; }

Один і той самий стиль можна одночасно застосувати до кількох елементів. Для цього необхідно в лівій частині оголошення перерахувати через кому потрібні селектори:

p, span { color: black; }
